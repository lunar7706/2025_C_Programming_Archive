# 📚 2025 C Programming Archive

> **본 저장소는 2025년도 1학기 IT학부 "C언어" 수업 학습 기록 및 과제와 개별 연습을 모아둔 아카이브 저장소입니다.**

## 📖 Key Study Themes

### 1. 효율적인 소스 코드 관리
- 여러 개의 레포지토리를 하나의 아카이브로 통합하여 **가시성**을 높였습니다.
- `git subtree` 명령어를 사용하여 원본 커밋 이력을 보존하며 관리합니다.

### 2. 알고리즘 및 데이터 구조
- **Number System:** 2진법, 10진법, 16진법 간의 상호 변환 로직 구현
- **Function modularization:** 사용자 정의 함수를 통한 코드 재사용성 극대화

---

## 🚀 How to Use
1. 각 폴더는 독립적인 컴파일이 가능한 소스 코드(`main.cpp` 등)를 포함하고 있습니다.
2. `Visual Studio Code`에서 해당 폴더를 열고 `g++` 또는 `gcc`를 통해 컴파일 후 실행할 수 있습니다.
